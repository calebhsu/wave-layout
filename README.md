# wave-layout
Wave layout for CraftML

### Install
    $ npm install wave-layout

### Parameters
- height: adjusts wave height
- spacing: adjusts spacing of item arrangement

### Example
```html
<craft>
    <craft name="wave-layout" module="wave-layout"/>
    
    <wave-layout>
        <repeat n="10">
            <cube></cube>
        </repeat>
    </wave-layout> 

</craft>
```
